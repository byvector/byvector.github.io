菱形图片

有时候可能会遇到菱形图片的需求，可以用下面两种方式实现。

基于变形的方案

这种方式与之前平行四边形的解决方案类似，需要把图片用一个div包裹，然后div和img进行相反的变形。

<div class="diamond-a">
    <img src="cat.jpg">
</div>
.diamond-a{
    margin: 100px;
    width: 250px;
    height:250px;
    transform: rotate(45deg);
    overflow: hidden;
}
.diamond-a > img{
    max-width: 100%;
    transform: rotate(-45deg);
}

伪随机背景

伪随机背景其实还是对条纹背景的扩展。假设一种场景，我们想得到不同颜色和不同宽度的垂直条纹。并且在背景平铺时不会产生接缝。利用linear-gradient好像可以轻松实现

<div class="rb-a"></div>
body{
    display: flex;
    flex-flow: row wrap;
    justify-content: space-around;
}
div{
    margin: 20px 20px;
    padding: 0;
    min-width: 20em;
    min-height: 6em;
}
.rb-a{
    background: linear-gradient(90deg,#fb3 15%,#655 0,#655 40%,#ab4 0,#ab4 65%,hsl(20,40%,90%) 0);
    background-size: 80px 100%;
}
