闭包

闭包，在JavaScript中很有意思。JavaScript是链式作用域，所以可以将函数内部与函数外部连接起来。由于在Javascript中，只有函数内部的子函数才能读取函数内部的变量，因此可以把内部的函数返回，通过返回的函数就能够读取到函数内部的变量。
于是，可以把闭包简单理解成定义在一个函数内部的函数，并且该内部函数访问了外部的变量。

console.log("-----闭包-----");
function fun(){
    var a = 1;
    return function f(){
        console.log(a);
    };
}
var result = fun()();
