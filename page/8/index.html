RegExp类型

RegExp 对象表示正则表达式，它是对字符串执行模式匹配的强大工具。有两种创建方式

  • 字面量
    var p = /pattern/attributes;
    比如：匹配第一个“bat”或“cat”，不区分大小写
    var pattern = /[bc]at/i;
  • RegExp构造函数
    var p = new RegExp(pattern,attributes);
    比如：匹配第一个“bat”或“cat”，不区分大小写
    var pattern = new RegExp("[bc]at","i");
